## 🐛 常见问题排查

### 问题1：图片不显示

**原因**：图片路径错误或文件未上传  
**解决方案**：确保 `images/` 文件夹包含以下文件：
- instructor-linwenguan.jpg
- instructor-yuanfeng.jpg
- instructor-chenshuke.jpg
- digital-human-platform.jpg
- digital-human-effect.jpg

### 问题2：新模块样式不生效

**原因**：new-modules.css未引入  
**解决方案**：确保HTML头部包含：

```html
<link rel="stylesheet" href="css/style.css">
<link rel="stylesheet" href="css/new-modules.css">
```

### 问题3：移动端布局错乱

**原因**：响应式断点未正确配置  
**解决方案**：检查CSS中的媒体查询，特别是：
- @media (max-width: 768px)
- @media (max-width: 480px)
